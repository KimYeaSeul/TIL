배열과 문자열에서 많이 사용하는 메소드

  • 자바스크립트에서 배열은 객체 => 속성과 메소드 가짐
  • 배열은 mutable(수정가능), 문자열은 immutable(수정불가)
  • 원래 형태를 변경시키지 않는 메소드는 배열과 문자열 둘 다 사용 가능

속성

.length
문자열과 배열의 길이를 잼

메소드

배열에서만 적용
push()
배열의 뒤 쪽에 새로운 원소를 삽입. 변수의 길이 1 증가
pop()
배열의 마지막 원소를 빼냄. 변수의 길이 1 감소
*push와 pop은 자료구조 stack과 유사
shift()
배열의 맨 앞에 새로운 값 삽입
unshift()
배열의 맨앞 원소를 빼냄
join()
배열을 합치는 기능, 문자열로 바꿀때 사용
slice(startIndex, endIndex)
배열을 startIndex부터 endIndex-1 까지 잘라서 새로운 배열을 만듬(기존 배열은 변하지 않음)
splice(startIndex, numElement)
배열을 startIndex부터 numElement 개수만큼 자름.(기존 배열 변함.)
문자열에서만 적용
splite()
문자열을 문자의 배열로 나누고 싶을 떄 사용
둘다 적용(원래의 형태 유지시켜 주는 것들)
concat()
합치는 기능
indexOf()
원소가 가지고 있는 인덱스를 앞에서부터 찾을 수 있음
lastIndexOf()
원소가 가지고 있는 인덱스를 뒤에서부터 찾을 수 있음
